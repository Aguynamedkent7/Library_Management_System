To run QRCodeGeneratorGUI.java
Your console needs to be on the same directory as the file
Then compile the file using this command "javac -cp ".;libs/core-3.5.1.jar;libs/javase-3.5.1.jar" QRCodeGeneratorGUI.java"
after compiling execute this command to run it "java -cp ".;libs/core-3.5.1.jar;libs/javase-3.5.1.jar" QRCodeGeneratorGUI"
